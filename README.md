## Global Install

> Clone repository

	npm i
	npm link

<b>Command:</b> 

	itec [presentation] -t [top] -b [bottom] -s [screens]

<b>Arguments:</b>

> path to presentation (empty folder)

	presentation

<b>Flags:</b> 

> Number of pixels to trim top and bottom of picture

	-t [top]
	-b [bottom]

> Path to folder with screens (can be `png` or `jpg`): *./screens* - by default

	-s [screens]
